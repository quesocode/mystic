# Mystic
Mystic is a native photo app for iOS. 
The goal behind the app is to live somewhere between Photoshop and Instagram: Easy to use, but powerful enough to create something truly original.
__Photo app enlightenment__

#### Features

* Font Support
* Sketching
* Brushes & Brush editing
* Layers
* Image Adjustments
  * Brightness
  * Contrast
  * Sharpness
  * Color Tone
  * Vignette (colored, opacity, position)
  * Haze Control
  * Saturation
  * Skin Tone Editing
  * Gamma Control
  * Exposure
  * Levels
  * RGB Control
  * Shadow / Highlights
* Predesigned Word Art & Designs
* Tools
  * Frames
  * Light Effects
  * Color Overlays
  * Sketching
    * 14 Preset Brushes
    * Brush Editing Controls
    * Create your own brushes
    * Bristle Density
    * Bristle Size Control
    * Intensity
    * Tilt
    * Spacing
    * Jitter & Scatter
    * Dynamic Tilt
    * Dynamic Weight
    * Dynamic Intensity
    * Eraser
    * Layers
    * Layer Fill
    * Desaturate Layer
    * Clear Layer
    * Brush Color
    * Brush Size
    * Brush Opacity
* 38 Preset Color Filters
* Categorised Textures to add to your photo
